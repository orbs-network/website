AI Agent Order Types

This MCP-compatible DeFi skill gives any AI coding agent (Claude Code, Cursor, Windsurf) and AI agents access to the full spectrum of on-chain swap order types — all gasless, non-custodial, and oracle-protected.

⚡

Market Orders

Instant gasless swaps at best available price with oracle protection.

🎯

Limit Orders

Set a target price — the order executes automatically when the market reaches it.

🛡️

Stop-Loss

Protect your position with automatic sell triggers below a price threshold.

📈

Take-Profit

Lock in gains by auto-selling when price rises above your target.

⏱️

TWAP / Chunked

Split large orders into time-weighted chunks with oracle protection on every fill.

How It Works

From user intent to on-chain execution — the skill handles the entire flow.

1

Define Intent

User or agent specifies: chain, tokens, amount, order type, and optional price constraints.

2

Wrap & ApproveOptional

If needed, wrap native tokens and approve the contract to spend your tokens. The skill provides the calldata.

3

Prepare Order

The skill generates approval calldata, EIP-712 typed data, and relay-ready submit payloads.

4

Sign & Submit

User signs with their wallet (EIP-712). The signed order is submitted to the decentralized relay.

5

Oracle-Protected Execution

Orbs L3 network executes with best-price routing and oracle protection on every chunk.
