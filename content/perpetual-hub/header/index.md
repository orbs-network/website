# Turn-key on-chain 
# perpetual futures platform
# for any venue or developer

Perpetual Hub Ultra is a fully integrated, chain-native perps trading protocol that any builder, frontend, or protocol can adopt and deploy, complete with a customizable user interface and modular components. Powered by Orbs' advanced Layer 3 infrastructure, Ultra unlocks seamless access to high-performance perpetual futures with minimal lift.

Perpetual Hub Ultra is built on top of Symm.io's smart contract infrastructure, which serves as the foundation for intent based perpetual trading. Its modular design allowed us to build a tailored implementation on top, combining execution logic, UI, and liquidity systems into a complete, ready to deploy solution.
